Strategic Fusion Dashboard

This dashboard brings OSINT, HUMINT, IMINT, and other intelligence records into one map-based workspace. It is built for rapid review, geospatial filtering, image inspection, and maintaining a common operating picture.

Back to Dashboard

Core Workflow

  1. Load data: Fetch existing MongoDB records or import CSV, JSON, Excel, and image files.
  2. Add nodes: Use the manual node form when you already know the location and details.
  3. Inspect nodes: Hover over markers for a quick preview or click a marker to open the dossier panel.
  4. Maintain records: Use dossier actions to edit or delete MongoDB-backed nodes.
  5. Filter the map: Search by keyword and narrow by intelligence type or source.

Supported Data

  • OSINT Open Source Intelligence
  • HUMINT Human Intelligence
  • IMINT Imagery Intelligence
  • Additional MongoDB-backed types such as SIGINT, CYBINT, and SEBINT
  • Imported records must include valid latitude and longitude values.

Map Tools

  • Markers are clustered when many nodes are close together.
  • The hover card gives a fast preview without opening the full dossier.
  • The legend shows intelligence types currently visible on the map.
  • Terrain Setup can pin a custom image overlay to real-world map bounds.

Node Actions

  • Edit Details: Updates title, type, coordinates, source, image URL, and description.
  • Delete Node: Opens a confirmation dialog before removing a record.
  • Success and error notifications appear in the center of the screen.
  • Update and delete require the node to have a MongoDB _id.

Current Limitations

  • AWS S3 integration is not implemented in this version.
  • File ingestion is local and browser-based.
  • Terrain overlay bounds must be entered manually.
  • Reverse geocoding uses OpenStreetMap Nominatim, so rate limits may apply.
  • Authentication and role-based access control are not included yet. Do not expose update/delete routes in a real deployment without access control.
  • This is a prototype dashboard and should not be used for operational intelligence decisions.

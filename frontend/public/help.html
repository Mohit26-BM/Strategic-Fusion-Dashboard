Intelligence Dashboard Help

This dashboard is a workspace for ingesting, visualizing, and filtering geospatial intelligence nodes. It is designed for rapid review, mission planning, and situational awareness.

How to Use

  1. Ingest Data: Use the File Upload or Add Node forms to add intelligence nodes. Supported types include OSINT, HUMINT, SIGINT, CYBINT, IMINT, and more.
  2. Filter & Search: Use the filter panel to search by type, source, or keywords. The map and list update instantly.
  3. Map & Overlay: Visualize nodes on the map. Use Terrain Setup to overlay mission images or custom maps.
  4. Review Details: Click a node to see its full dossier and metadata.

Features

  • Bulk and manual data ingestion (with validation)
  • Geospatial filtering and search
  • Custom terrain overlays (mission images, satellite, etc.)
  • Legend and type color-coding
  • MongoDB backend for persistence

Limitations

  • Overlay alignment requires manual bounds entry
  • Reverse geocoding uses free OpenStreetMap API (rate limits apply)
  • No authentication or access control (demo only)
  • Not for operational use—demo/prototype only
  • Browser local storage used for some settings
